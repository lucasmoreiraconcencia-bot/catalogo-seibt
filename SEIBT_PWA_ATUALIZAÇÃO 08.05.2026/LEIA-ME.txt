# SEIBT — Catálogo PWA

## Arquivos do PWA
- `index.html` → app principal
- `manifest.json` → configuração do PWA
- `sw.js` → funcionamento offline
- `icon-192.png` / `icon-512.png` → ícones do app
- `logo.png` → logo original (não é obrigatório, pode remover)

## Como hospedar (grátis e fácil)

### Opção 1: Netlify (recomendada — 2 minutos)
1. Acesse: https://app.netlify.com/drop
2. Faça login (Google ou GitHub)
3. **Arraste a pasta inteira `pwa`** para a área indicada
4. Pronto! Vai gerar um link tipo `https://nome-aleatorio.netlify.app`
5. Esse link você manda pros representantes

### Opção 2: GitHub Pages
- Suba os arquivos num repositório → Settings → Pages → Branch main

## Como o representante usa
1. Abre o link no celular
2. Chrome (Android): aparece "Adicionar à tela inicial"
   iPhone (Safari): toca em Compartilhar → "Adicionar à Tela de Início"
3. O app fica como ícone no celular igual qualquer outro app
4. Funciona offline depois da primeira abertura

## Como você (admin) atualiza fotos
1. Toca em **🔒 Admin** (canto superior direito)
2. Digita a senha: `MGHS1500A2#`
3. Aparece um botão de câmera 📷 em cada card
4. Toca no botão → escolhe foto da galeria ou tira na hora
5. Foto sobe automaticamente pra nuvem (ImgBB) e aparece pra todos

## Notas
- As fotos ficam salvas no servidor ImgBB com sua chave de API
- Cada representante vê as fotos automaticamente após você atualizar
- Pra trocar a senha, edite no `index.html` a linha: `const ADMIN_PASSWORD = 'MGHS1500A2#';`
